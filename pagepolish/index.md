# PagePolish

**The ultimate formatting assistant for Confluence Cloud.**

PagePolish automatically detects and resolves common formatting issues in your Confluence pages, ensuring a professional and consistent look with just one click.

## Key Features
*   **Remove Empty Lines**: Clean up excessive whitespace.
*   **Fix Detached Bullets**: Reconnect orphaned list items.
*   **Horizontal Rules**: Manage or remove divider lines.
*   **One-Click Clean**: Apply all fixes instantly.
